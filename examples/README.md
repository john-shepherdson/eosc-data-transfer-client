# EOSC Data Transfer Python Client Examples

This directory contains simple example scripts that illustrate how the eosc-data-trasnfer-client library can be used.

To run the scripts run the following commands:
```bash
git clone https://gitlab.com/fts/eosc-data-transfer-client.git
cd eosc-data-transfer-client
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
pip install -e .
python3 examples/example.py
```
